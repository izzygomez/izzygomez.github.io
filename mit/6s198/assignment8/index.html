Sleepy

and here's a link to the Sleepy app source.

In the Sleepy application, I created a simple user interface with three buttons which start a timer of 20, 60, or 90 minutes, respectively. The timers are created by the use of the implicit intent described by the ACTION_SET_TIMER action (as explained here). These values represent the duration of naps to take throughout the day, in accordance with what research suggest are optimal nap time durations; the inspiration/source for the application is linked from the app (and here!).

GetBackToMe

Here's a link to the GetBackToMe app *.apk,

and here's a link to the GetBackToMe app source.

In the GetBackToMe application, I again created a relatively simple UI that gives a user two options: to select a contact, and to send that contact an SMS message. When selecting a contact, the implicit intent described by the ACTION_PICK action is used, and the data is received back in the application for manipulation (using startActivityForResult()). In this app, we extract the contact's phone number, and then give the user the option to send them a pre-defined text message that let's their contact know to get back to the sender as soon as possible (i.e. possibly tried to contact them, but they didn't pick up, so you use this app to send this pre-defined message instead of always having to manually retype the message in a messaging app).
